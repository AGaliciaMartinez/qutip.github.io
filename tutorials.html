Tutorials


This pages contains our collection of IPython notebooks for introducing and demonstrating features of QuTiP.

Example notebooks

These notebooks demonstrates basic functionality in QuTiP. Going through these notebooks should be a good way to get familiarized with the software.

Quantum mechanics lectures with QuTiP

These lecture-style notebooks focus on a particular quantum mechanics problem and analyze them numerically using QuTiP.

Development notebooks

These development notebooks are somewhat more technical and focuses on testing and benchmarking new features of QuTiP.

  • development-ipynbtools-test

    Tests and demonstrations of the IPython notebook tools provided by QuTiP.

  • development-mesolve-and-liouvillians

    Tests and demonstrate various ways to specify a master equation using QuTiP's mesolve function.

  • development-correlation-functions

    Tests and comparisons between different correlation functions solvers in QuTiP.

  • development-liouvillian-construction

    Tests and benchmarks of the performance of QuTiP's functions liouvillian and liouvillian_fast.

  • development-floquet-test

    Tests and comparisons between the Floquet solver and time-dependent master equation for period time-dependent problems.

  • development-mcf90-test

    A series of test problems for the Fortran90 version of the Monte-Carlo wave function solver.
